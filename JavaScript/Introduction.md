# JavaScript简介

```Javascript``` 是一种轻量级解释型脚本语言可以用来创建动态更新的内容，控制多媒体，制作2D/3D动画等.

?> 解释型：运行之前不会被编译也不会报错二级，代码在执行之前需要转换为其他形式

?> 编译型：对源码进行编译，还能检查语法错误，比如C/C++先被编译成汇编语言然后以二进制的格式运行。

## JS的组成部分
- 核心(ECMAScript)定义了JavaScript的基本语法和数据类型
- 文档对象模型(DOM,Document Object Model) 让JS可以与网页进行对话，例如修改html的节点
- 浏览器对象模型(BOM,Browser Object Model) 让JS可以与浏览器进行交互，例如改变浏览窗口的大小
